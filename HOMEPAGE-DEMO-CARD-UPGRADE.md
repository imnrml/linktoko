# LinkToko Homepage Demo Card Upgrade

Update ini merapikan bagian demo dan template card di halaman utama.

Yang berubah:
- Section demo sekarang menampilkan 4 kategori: Kuliner, Fashion, Jasa, Digital.
- Copywriting demo dibuat lebih menjual dan lebih jelas.
- Ada CTA baru: "Mau punya toko seperti demo di atas?"
- Card template di bagian marketplace punya deskripsi lebih lengkap dan tag fitur.
- Demo di folder `demos/` sudah ikut versi dynamic premium.

Cara upload:
1. Extract ZIP.
2. Upload/replace semua isi folder ke root repo GitHub `linktoko`.
3. Commit changes.
4. Tunggu 1-3 menit.
5. Refresh website dengan Ctrl + F5.
