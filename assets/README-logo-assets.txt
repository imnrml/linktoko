- linktoko-logo-luxury-horizontal.png

- linktoko-logo-luxury-horizontal.svg

Cara pakai di GitHub:
1. Upload semua file ini ke folder `assets/` di repo `linktoko`.
2. Untuk logo header, pakai:
   `assets/linktoko-logo-luxury-horizontal-transparent.png`
3. Untuk icon/favicon, pakai:
   `assets/linktoko-icon-luxury.png`

Kalau mau ganti logo di header website:
- Buka `index.html`
- Cari bagian logo/nav brand
- Ganti dengan img:
  <img src="assets/linktoko-logo-luxury-horizontal-transparent.png" alt="LinkToko" class="brand-logo">
