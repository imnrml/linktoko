# LinkToko Landing Page Upgrade

Versi ini menambahkan elemen visual dan trust agar landing page lebih siap dipakai promosi:

- Hero baru dengan mockup HP dan gambar produk.
- Section preview visual untuk kuliner, fashion, dan jasa.
- Section cara order 3 langkah.
- Section keunggulan dengan icon.
- Template card memakai gambar.
- Harga paket lebih jelas.
- FAQ.
- Testimoni awal yang bisa diganti nanti.
- CTA akhir untuk isi form order dan konsultasi WhatsApp.

## Cara upload ke GitHub

1. Extract file ZIP.
2. Masuk folder `linktoko-mvp`.
3. Upload/replace semua isi folder ke repo GitHub `linktoko`.
4. Klik Commit changes.
5. Tunggu 1-3 menit.
6. Buka `https://imnrml.github.io/linktoko/` lalu refresh dengan Ctrl+F5.

## Catatan gambar

Gambar di landing page memakai URL gambar online gratis dari Unsplash. Kalau nanti ingin pakai gambar sendiri, ganti nilai `src="..."` di `index.html` dan `image: '...'` di `script.js`.
