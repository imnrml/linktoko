# Cara update website LinkToko di GitHub

1. Buka repo GitHub `linktoko`.
2. Upload/replace file ini dari folder update:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Klik `Commit changes`.
4. Tunggu 1-3 menit.
5. Buka ulang https://imnrml.github.io/linktoko/
