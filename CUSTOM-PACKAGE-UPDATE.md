# LinkToko Custom Package Update

Update ini menambahkan section custom, add-on, FAQ, dan detail paket yang lebih jelas.

## Yang berubah
- Paket Starter/Pro/Agency sekarang punya batas produk/menu dan revisi.
- Ada section Bisa Custom untuk menjelaskan customer boleh request nama toko, logo, gambar, menu, harga, kontak, dan sosmed.
- Ada section Add-on agar request tambahan bisa dihitung sebagai layanan berbayar.
- Ada FAQ supaya customer paham bahwa edit toko saat ini dibantu admin, belum melalui dashboard customer.
- CTA checkout diarahkan ke Google Form order.
